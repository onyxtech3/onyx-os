# Welcome to Onyx OS!

## About the OS

* This OS is based on Ubuntu.
* It has the Snap and Flatpak (if you don't know what those are, they are both app stores on Ubuntu).
* It comes preloaded with Steam, Google Chrome, Spotify and VS Code.
* I deleted some of the bloat with Ubuntu.
